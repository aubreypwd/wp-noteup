Take simple notes on your Posts and Pages.

== Description ==

WP NoteUp allows you to take simple notes when you're editing your Posts
or Pages.

== Installation ==

Install by going to your WordPress Dashboard > Plugins > Add New and searching
for "WP NoteUp."
